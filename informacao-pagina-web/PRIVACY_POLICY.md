# Política de Privacidade - Analisador de Página Web

**Última atualização:** 21 de Janeiro de 2026

A privacidade dos nossos usuários é uma prioridade. Esta política descreve como a extensão "Analisador de Página Web" coleta, usa e protege suas informações.

## 1. Coleta de Dados

A extensão coleta e processa as seguintes informações apenas quando ativada pelo usuário em uma aba específica:

- **Metadados da Página:** Título, descrição, palavras-chave, autor.
- **Conteúdo Técnico:** Cabeçalhos, links, scripts, tecnologias detectadas.
- **Dados do Navegador:** User Agent, resolução de tela, configurações de idioma.

## 2. Uso dos Dados

Todas as informações coletadas são processadas **localmente** no seu navegador.

- A extensão **não** envia dados para servidores externos.
- A extensão **não** armazena dados de navegação permanentemente.
- Os dados são exibidos apenas para o usuário no momento da análise.

## 3. Permissões

A extensão solicita as seguintes permissões para funcionar:

- **activeTab:** Para analisar o conteúdo da aba atual quando você clica no ícone.
- **scripting:** Para executar o script de análise na página.
- **tabs:** Para monitorar a URL atual e atualizar o ícone de status (On/Off) da extensão.

## 4. Compartilhamento de Dados

Nós não vendemos, trocamos ou transferimos suas informações para terceiros. A única forma de os dados saírem do seu navegador é se você utilizar explicitamente a função "Exportar JSON" ou "Gerar Relatório", o que cria um arquivo local no seu dispositivo.

## 5. Segurança

A segurança dos seus dados é importante para nós. Como esta extensão opera localmente, não mantemos bancos de dados de usuários. O código é aberto e auditável para garantir que nenhuma transmissão oculta de dados ocorra.

## 6. Alterações

Podemos atualizar esta política de privacidade ocasionalmente. As alterações serão refletidas na data de "Última atualização" no topo desta página.

## 7. Contato

Se tiver dúvidas sobre esta política, entre em contato através da página de suporte na Chrome Web Store ou pelo repositório do projeto.
